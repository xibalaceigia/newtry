扫拖一体机器人维修保养指南（完整版）
================================

【保养总则】
1. 定期保养可延长机器寿命2-3年。
2. 每次使用后做基础清洁，每周做深度保养。
3. 使用原厂或认证配件，劣质配件会损坏机器。
4. 保养前务必断电、关机、取出电池（如可拆卸）。
5. 机身不可泡水，只能用微湿布擦拭。

【每日保养（每次使用后）】
6. 倒空尘盒：每次清扫后倒掉垃圾，避免细菌滋生。
7. 清理主刷：用附赠的清洁刀割断缠绕的头发。
8. 清理边刷：取下边刷，清除根部缠绕的毛发。
9. 检查拖布：如有明显污渍，手洗或机洗（可水洗拖布）。
10. 擦拭传感器：用干软布轻擦底部悬崖传感器和前端红外传感器。
11. 检查充电触点：观察有无氧化发黑，用干布擦拭。

【每周保养】
12. 清洗滤网：可水洗滤网用清水冲洗，不可水洗的轻敲除尘。
13. 晾干滤网：洗后至少晾24小时，完全干透才能装回。
14. 清洗拖布：温水加中性洗涤剂手洗，或洗衣机轻柔模式。
15. 清理万向轮：拆下万向轮，清除轮轴缠绕的头发和灰尘。
16. 擦拭激光雷达：用软布轻擦顶部激光雷达窗口，勿用酒精。
17. 清洁碰撞缓冲器：擦拭前挡板，确保按压回弹正常。
18. 检查主刷盖板：确认卡扣无断裂，盖板闭合紧密。
19. 清理吸尘口：用手指或镊子取出吸尘口深处的异物。
20. 清洁充电座：擦拭充电座触点及红外发射窗。

【每月保养】
21. 深度清洁主刷：拆下主刷两端轴承盖，清除内部缠发。
22. 清理驱动轮：用镊子清除轮胎花纹中嵌入的杂物。
23. 检查轮子磨损：胎纹磨平会导致打滑，需更换驱动轮总成。
24. 清理边刷轴：边刷取下后，清洁轴心和卡扣处的污垢。
25. 擦拭机身外壳：用微湿软布擦拭，避免水流入缝隙。
26. 检查电池触点：确保电池接口无氧化、无松动。
27. 清理尘盒密封条：密封条老化会导致漏风吸力下降。
28. 检查拖布支架：确认卡扣和弹簧功能正常。

【基站保养（如有）】
29. 每周倒污水箱：每次拖地后倒掉污水，冲洗箱内。
30. 每周清洗污水箱滤网：拆下滤网，冲洗掉毛发和污垢。
31. 每周擦拭基站底盘：用刷子或湿布清理积水、污垢。
32. 每周清洗拖布清洗肋：取出清洗肋，冲洗底部沉淀物。
33. 每月清理基站排污口：用棉签清理细小杂质。
34. 每月检查清水箱滤网：防止杂质进入水路。
35. 每月清理基站干燥模块：用干布擦拭出风口。
36. 基站除臭：污水箱加少量白醋或官方除臭模块。
37. 基站防霉：保持基站盖板开启通风，开启烘干功能。
38. 检查上下水版管路（如有）：确认无漏水、无堵塞。

【配件更换周期】
39. 边刷：3-6个月，刷毛变形或脱落即换。
40. 主刷：6-12个月，刷毛磨损或刮条断裂即换。
41. 滤网：3-6个月，洗过几次后过滤效果下降即换。
42. 拖布：2-3个月，发硬、发黑、异味难除即换。
43. 电池：2-3年，续航明显下降或鼓包即换。
44. 尘袋（集尘基站）：2-3个月，满了或破损即换。
45. 主刷盖板：卡扣断裂无法闭合即换。
46. 万向轮：转动不畅或磨损严重时更换。
47. 驱动轮：胎纹磨平或电机异响时更换。
48. 激光雷达模组：扫描异常或报错时送修更换。
49. 传感器：感应失灵时更换。
50. 充电座主板：无法充电且触点正常时送修。

【故障诊断与维修】

【充电与电源类】
51. 故障：完全不充电
    排查：检查充电座电源 → 清洁金属触点 → 检查适配器 → 更换电池
52. 故障：充不进电（充很久只有一格）
    排查：电池老化 → 更换电池
53. 故障：充电时发热严重
    排查：电池损坏 → 立即停止充电 → 更换电池
54. 故障：一充电就报错
    排查：主板充电电路故障 → 送修
55. 故障：电池鼓包
    排查：立即停用 → 更换电池（有起火风险）

【运行与导航类】
56. 故障：不开机
    排查：充电30分钟 → 长按电源键10秒强制重启 → 检查电池接口
57. 故障：开机后立刻关机
    排查：电池电量耗尽 → 电池损坏 → 主板短路
58. 故障：原地打转
    排查：清洁悬崖传感器 → 检查驱动轮是否卡死 → 轮子磨损打滑
59. 故障：走歪或画圈
    排查：陀螺仪故障（低端机） → 轮子转速不同步 → 送修
60. 故障：乱撞墙
    排查：清洁碰撞传感器 → 检查缓冲器是否卡住 → 红外传感器脏污
61. 故障：无法回充
    排查：清洁充电座和机器人红外窗 → 移动充电座位置 → 检查回充传感器
62. 故障：一直提示“悬崖传感器异常”
    排查：擦拭传感器 → 搬到另一房间测试 → 传感器损坏
63. 故障：激光雷达不转
    排查：手动拨动检查卡死 → 清理灰尘 → 激光模块故障送修
64. 故障：建图失败或地图错乱
    排查：重新建图 → 清理激光雷达 → 恢复出厂设置

【清扫与吸力类】
65. 故障：吸力变小
    排查：倒空尘盒 → 清洗滤网 → 清理吸尘口堵塞 → 检查密封条
66. 故障：主刷不转
    排查：清理缠绕物 → 检查主刷电机 → 更换主刷总成
67. 故障：边刷不转
    排查：清理缠绕物 → 检查齿轮箱 → 更换边刷电机
68. 故障：异响（嘎嘎声）
    排查：主刷卡异物 → 边刷变形 → 轮子进石子 → 风机叶轮损坏
69. 故障：异响（尖锐声）
    排查：轴承缺油 → 电机故障 → 送修
70. 故障：尘盒盖关不严
    排查：密封条变形 → 卡扣断裂 → 更换尘盒

【拖地类故障】
71. 故障：拖地不出水
    排查：水箱无水 → 气孔堵塞 → 水泵故障 → 电控板问题
72. 故障：出水不停（水渍严重）
    排查：水箱密封圈老化 → 水泵常通 → 电控故障
73. 故障：拖布洗不干净
    排查：清洗肋堵塞 → 清水水量不足 → 清洗盘故障
74. 故障：拖布烘干不干
    排查：烘干模块故障 → 环境温度过低 → 出风口堵塞
75. 故障：基站漏水
    排查：清水箱破损 → 管路脱落 → 水泵密封老化 → 停用送修
76. 故障：污水箱不抽水（上下水版）
    排查：污水管堵塞 → 水泵故障 → 水位传感器失灵

【连接与智能类】
77. 故障：无法连接Wi-Fi
    排查：确认2.4G频段 → 重启路由器 → 关闭手机VPN → 重启机器
78. 故障：App显示离线
    排查：机器人是否开机 → 路由器信号弱 → 休眠唤醒
79. 故障：定时任务不执行
    排查：检查系统时间 → 重新设置任务 → 恢复出厂设置
80. 故障：固件升级失败
    排查：网络稳定 → 保持电量>50% → 重启后重试 → 联系售后
81. 故障：地图丢失
    排查：基站被移动 → 意外关机 → 重新建图

【硬件更换教程（简易版）】

82. 更换边刷：
    - 拔出旧边刷 → 对准D型轴按下新边刷 → 听到咔哒声即可

83. 更换主刷：
    - 按下卡扣打开盖板 → 取出旧主刷 → 取出两端轴承盖换到新主刷 → 装回

84. 更换滤网：
    - 打开尘盒 → 抽出旧滤网 → 插入新滤网 → 注意方向不要装反

85. 更换拖布：
    - 撕下旧拖布（魔术贴） → 贴上新拖布 → 抚平确保贴合

86. 更换电池（以常见机型为例）：
    - 拆下底部螺丝 → 打开底盖 → 拔下电池插头 → 换新电池 → 复原
    - 注意：部分机型需焊接，建议专业人员操作

87. 更换万向轮：
    - 撬出旧轮子 → 按压卡扣拔下 → 插入新轮子 → 咔哒声到位

88. 更换充电座触点：
    - 卸下充电座背面螺丝 → 拆开外壳 → 拔出触点组件 → 换新 → 装回

【清洁剂与工具推荐】
89. 推荐工具：附赠清洁刀、十字/六角螺丝刀套装、镊子、软毛刷、气吹
90. 推荐清洁剂：中性洗涤剂、白醋（除臭）、官方专用清洁液
91. 禁止使用：酒精、84消毒液、强酸强碱、钢丝球（刮伤传感器）

【长期存放与季节保养】
92. 长期不用（>1个月）：
    - 清洁干净所有部件 → 晾干拖布和水箱 → 充满电后关机 → 取出电池（如可拆卸） → 放阴凉干燥处
93. 回南天/梅雨季：
    - 每天倒尘盒 → 开启拖布烘干 → 污水箱每次倒空 → 基站盖板留缝通风
94. 冬季低温（<0℃）：
    - 水箱不可留水（防结冰胀裂） → 电池避免低温充电 → 尽量在室温环境使用
95. 搬家运输：
    - 取出电池 → 清空水箱尘盒 → 固定激光雷达（用胶带） → 原箱包装最佳

【报修与售后】
96. 保修期内故障：联系官方售后，不要自行拆机（可能失去保修）
97. 常见需送修的故障：
    - 激光雷达不转/报错
    - 主板进水
    - 驱动轮电机异响
    - 水泵故障
    - 充电电路烧毁
98. 送修前准备：备份地图数据（部分App支持）、清空尘盒水箱、拍照记录机身SN码
99. 保养记录建议：在手机备忘录记录每次更换配件的时间和日期
100. 终极建议：平时保养得当，送修概率降低80%
